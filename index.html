Aquisição de dados

Este é o processo de conectar seus dados de onde quer que estejam, no Power BI e, em seguida, moldá-los para atender às suas necessidades. A ferramenta revolucionária do Power BI chamada Power Query será fundamental para ajudar você a fazer isso.

Modelagem de Dados

Este é o processo de selecionar seus dados e transformá-los em algo mais útil do que apenas a versão bruta. O revolucionário Power Pivot do Power BI ajudará você a fazer isso. Depois de aprimorar suas habilidades e aprender a utilizar o Power Pivot, você estará no caminho certo para se tornar proficiente em Power BI.

Visualização de dados

Depois de carregar seus dados com sucesso e ter o banco de dados projetado para responder às perguntas de negócios desejadas, você precisará visualizar os dados. O novo mecanismo de visualização do Power BI está em um estado de melhoria contínua, mas com pouca orientação você deve começar a trabalhar.
